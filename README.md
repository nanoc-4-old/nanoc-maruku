# nanoc-maruku

This provides an [Maruku](http://www.methods.co.nz/maruku/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-maruku`

## Usage

```ruby
filter :maruku
```

This filter takes no options.
