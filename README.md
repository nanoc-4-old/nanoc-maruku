# nanoc-maruku

This provides a [Maruku](http://maruku.rubyforge.org/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-maruku`

## Usage

```ruby
filter :maruku
```

Parameters passed to this filter will be passed on to Maruku.
